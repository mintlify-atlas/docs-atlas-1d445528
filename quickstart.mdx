## Get Started in 5 Minutes

This guide will help you quickly set up and make your first API call to the Album Collection Manager.

<Steps>
  <Step title="Clone the Repository">
    Clone the project from GitHub:
    
    ```bash
    git clone https://github.com/elCorbacho/18.web2-examen
    cd 18.web2-examen
    ```
  </Step>

  <Step title="Run the Application">
    Use the included Maven wrapper to start the application:
    
    ```bash
    ./mvnw spring-boot:run
    ```
    
    <Note>
      The application will start on `http://localhost:8080` and automatically initialize an H2 in-memory database with sample data.
    </Note>
    
    You should see output similar to:
    
    ```
    Started Application in 3.456 seconds (JVM running for 4.123)
    ```
  </Step>

  <Step title="Make Your First API Call">
    Test the API with a simple GET request to retrieve all albums:
    
    <CodeGroup>
    ```bash curl
    curl http://localhost:8080/api/albums
    ```
    
    ```javascript JavaScript
    fetch('http://localhost:8080/api/albums')
      .then(response => response.json())
      .then(data => console.log(data));
    ```
    
    ```python Python
    import requests
    
    response = requests.get('http://localhost:8080/api/albums')
    print(response.json())
    ```
    </CodeGroup>
    
    Expected response:
    
    ```json
    {
      "status": "success",
      "timestamp": "2026-03-12T10:30:00",
      "message": "Álbumes recuperados exitosamente",
      "data": {
        "content": [],
        "totalElements": 0,
        "totalPages": 0,
        "pageNumber": 0,
        "pageSize": 10
      }
    }
    ```
  </Step>

  <Step title="Create Your First Album">
    Create a new album using a POST request:
    
    <CodeGroup>
    ```bash curl
    curl -X POST http://localhost:8080/api/albums \
      -H "Content-Type: application/json" \
      -d '{
        "nombre": "World Cup 2026",
        "year": 2026,
        "descripcion": "Official FIFA World Cup sticker album"
      }'
    ```
    
    ```javascript JavaScript
    fetch('http://localhost:8080/api/albums', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({
        nombre: 'World Cup 2026',
        year: 2026,
        descripcion: 'Official FIFA World Cup sticker album'
      })
    })
    .then(response => response.json())
    .then(data => console.log(data));
    ```
    
    ```python Python
    import requests
    
    album_data = {
        "nombre": "World Cup 2026",
        "year": 2026,
        "descripcion": "Official FIFA World Cup sticker album"
    }
    
    response = requests.post(
        'http://localhost:8080/api/albums',
        json=album_data
    )
    print(response.json())
    ```
    </CodeGroup>
    
    Expected response:
    
    ```json
    {
      "status": "success",
      "timestamp": "2026-03-12T10:35:00",
      "message": "Álbum creado exitosamente",
      "data": {
        "id": 1,
        "nombre": "World Cup 2026",
        "year": 2026,
        "descripcion": "Official FIFA World Cup sticker album",
        "active": true,
        "createdAt": "2026-03-12T10:35:00",
        "updatedAt": "2026-03-12T10:35:00"
      }
    }
    ```
  </Step>
</Steps>

## Common Issues

<AccordionGroup>
  <Accordion title="Port 8080 is already in use">
    If port 8080 is already occupied, you can change it by setting the `server.port` property:
    
    ```bash
    ./mvnw spring-boot:run -Dspring-boot.run.arguments=--server.port=8090
    ```
  </Accordion>
  
  <Accordion title="Maven wrapper not executable">
    On Unix-based systems, make sure the Maven wrapper has execute permissions:
    
    ```bash
    chmod +x mvnw
    ```
  </Accordion>
  
  <Accordion title="Java version error">
    This API requires Java 21 or higher. Check your version:
    
    ```bash
    java -version
    ```
    
    If you need to install Java 21, visit [Adoptium](https://adoptium.net/).
  </Accordion>
</AccordionGroup>
