## Welcome to Album Collection Manager API

The Album Collection Manager API is a comprehensive Spring Boot REST service designed for managing collectible albums and their stickers (laminas). Built with modern Java 21 and Spring Boot 3.5, it provides robust features for collectors to track their collections with intelligent validation and duplicate detection.

## API Highlights

The API provides comprehensive endpoints organized into logical groups:

- **Albums**: Create, read, update, and soft delete album collections
- **Catalog**: Define the master list of available stickers for each album
- **Laminas**: Manage individual stickers with duplicate detection and bulk upload
- **Additional Resources**: Access supplementary data like Chilean cities, exoplanets, and more

<Note>
All API responses follow a consistent `ApiResponseDTO` wrapper pattern with success indicators, messages, and error details for predictable client integration.
</Note>

## Architecture Patterns

This API follows industry best practices:

- **Layered MVC Architecture**: Clean separation between controllers, services, and repositories
- **DTOs & Mappers**: Request/response transformation for API contract stability
- **JPA Auditing**: Automatic tracking of creation and modification timestamps
- **Global Exception Handling**: Centralized error handling with meaningful error codes
- **Validation Framework**: Jakarta Bean Validation for request validation
