---
title: "Get Album Summary"
description: "Retrieve summary statistics for an album including catalog and collection status"
---

## Endpoint

```
GET /api/albums/{id}/summary
```

Retrieves comprehensive summary statistics for an album, including total catalog size, owned stickers, repeated stickers, and missing stickers.

## Path Parameters

<ParamField path="id" type="long" required>
  The ID of the album to retrieve summary for
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable response message
</ResponseField>

<ResponseField name="data" type="object">
  Album summary data
  
  <Expandable title="AlbumSummaryDTO">
    <ResponseField name="id" type="long">
      Album unique identifier
    </ResponseField>
    
    <ResponseField name="nombre" type="string">
      Album name
    </ResponseField>
    
    <ResponseField name="year" type="integer">
      Album release year
    </ResponseField>
    
    <ResponseField name="active" type="boolean">
      Whether the album is active
    </ResponseField>
    
    <ResponseField name="totalCatalogo" type="integer">
      Total number of stickers in the catalog
    </ResponseField>
    
    <ResponseField name="totalLaminas" type="integer">
      Total number of owned stickers (including duplicates)
    </ResponseField>
    
    <ResponseField name="laminasRepetidasTotal" type="integer">
      Total count of repeated stickers
    </ResponseField>
    
    <ResponseField name="laminasFaltantesTotal" type="integer">
      Number of stickers still missing from the collection
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="timestamp" type="datetime">
  Response timestamp
</ResponseField>

## Example Request

```bash cURL
curl -X GET http://localhost:8080/api/albums/1/summary \
  -H "Content-Type: application/json"
```

## Calculation Examples

### Completion Percentage
```
completion = ((totalCatalogo - laminasFaltantesTotal) / totalCatalogo) * 100
           = ((100 - 40) / 100) * 100
           = 60%
```

### Unique Stickers Owned
```
uniqueStickers = totalLaminas - laminasRepetidasTotal
               = 75 - 15
               = 60
```
