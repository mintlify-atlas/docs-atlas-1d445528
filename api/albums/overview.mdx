## Introduction

The Albums API allows you to manage album records including creating, retrieving, updating, and deleting albums. All endpoints are prefixed with `/api/albums`.

## Base URL

```
/api/albums
```

## Response Format

All responses are wrapped in a standard `ApiResponseDTO` structure:

```json
{
  "success": true,
  "message": "Operation completed successfully",
  "data": { /* endpoint-specific data */ },
  "timestamp": "2026-03-12T10:30:00"
}
```

### Error Response Format

```json
{
  "success": false,
  "message": "Error description",
  "errorCode": "ERROR_CODE",
  "errors": { /* field-specific errors */ },
  "timestamp": "2026-03-12T10:30:00"
}
```

## Common Status Codes

| Code | Description |
|------|-------------|
| 200  | Success |
| 201  | Created successfully |
| 400  | Bad request - validation error |
| 404  | Album not found |
| 500  | Internal server error |
