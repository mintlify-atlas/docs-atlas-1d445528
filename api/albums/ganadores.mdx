---

## Endpoint

```
GET /api/albums/{albumId}/ganadores
```

Retrieves the list of winners (ganadores) associated with a specific album, ordered by year in descending order.

## Path Parameters

<ParamField path="albumId" type="long" required>
  The ID of the album to retrieve winners for
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable response message
</ResponseField>

<ResponseField name="data" type="array">
  Array of winner objects
  
  <Expandable title="GanadorAlbumDTO">
    <ResponseField name="artista" type="string">
      Artist or winner name
    </ResponseField>
    
    <ResponseField name="premio" type="string">
      Award or prize name
    </ResponseField>
    
    <ResponseField name="anio" type="integer">
      Year the award was won
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="timestamp" type="datetime">
  Response timestamp
</ResponseField>

## Example Request

```bash cURL
curl -X GET http://localhost:8080/api/albums/1/ganadores \
  -H "Content-Type: application/json"
```

## Response Details

<Note>
Winners are returned in **descending order by year** (most recent first).
</Note>

### Fields Description

- **artista**: The name of the artist or winner
- **premio**: The award or prize that was won
- **anio**: The year the award was won

## Example Usage

<Tabs>
  <Tab title="JavaScript">
    ```javascript
    async function getAlbumWinners(albumId) {
      const response = await fetch(
        `http://localhost:8080/api/albums/${albumId}/ganadores`
      );
      const data = await response.json();
      
      if (data.success) {
        console.log(`Found ${data.data.length} winners`);
        data.data.forEach(winner => {
          console.log(`${winner.anio}: ${winner.artista} - ${winner.premio}`);
        });
      }
    }
    
    getAlbumWinners(1);
    ```
  </Tab>
  
  <Tab title="Python">
    ```python
    import requests
    
    def get_album_winners(album_id):
        response = requests.get(
            f'http://localhost:8080/api/albums/{album_id}/ganadores'
        )
        data = response.json()
        
        if data['success']:
            print(f"Found {len(data['data'])} winners")
            for winner in data['data']:
                print(f"{winner['anio']}: {winner['artista']} - {winner['premio']}")
    
    get_album_winners(1)
    ```
  </Tab>

    ```java
    public List<GanadorAlbumDTO> getAlbumWinners(Long albumId) {
        String url = "http://localhost:8080/api/albums/" + albumId + "/ganadores";
        
        ResponseEntity<ApiResponseDTO<List<GanadorAlbumDTO>>> response = 
            restTemplate.exchange(
                url,
                HttpMethod.GET,
                null,
                new ParameterizedTypeReference<ApiResponseDTO<List<GanadorAlbumDTO>>>() {}
            );
        
        return response.getBody().getData();
    }
    ```
