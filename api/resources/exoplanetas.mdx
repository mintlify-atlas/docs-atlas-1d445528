## List All Exoplanets

<RequestExample>
```bash cURL
curl -X GET "http://localhost:8080/api/exoplanetas?page=0&size=10" \
  -H "Content-Type: application/json"
```
</RequestExample>

<ResponseExample>
```json 200

  "status": "success",
  "message": "Exoplanetas obtenidos exitosamente",
  "data": {
    "content": [
      {
        "id": 1,

        "tipo": "Súper Tierra",

        "masaRelativaJupiter": 2.34,
        "descubiertoAnio": 2015,
```

## Create Exoplanet

<RequestExample>
```bash cURL
curl -X POST http://localhost:8080/api/exoplanetas \
  -H "Content-Type: application/json" \
  -d '{
    "nombre": "Kepler-442b",
    "tipo": "Súper Tierra",
    "distanciaAnosLuz": 1206.0,
    "masaRelativaJupiter": 2.34,
    "descubiertoAnio": 2015
  }'
```
</RequestExample>

<ResponseExample>
```json 201
{
  "status": "created",
  "message": "Exoplaneta creado exitosamente",
  "data": {
    "id": 1,
    "nombre": "Kepler-442b",
    "tipo": "Súper Tierra",
    "distanciaAnosLuz": 1206.0,
    "masaRelativaJupiter": 2.34,
    "descubiertoAnio": 2015,
    "createdAt": "2026-03-12T10:00:00",
    "updatedAt": "2026-03-12T10:00:00"
  },
  "timestamp": "2026-03-12T10:30:00"
}
```

```json 400
{
  "status": "error",
  "message": "Datos de entrada inválidos",
  "data": null,
  "timestamp": "2026-03-12T10:30:00"
}
```
</ResponseExample>

### Request Body

<ParamField body="nombre" type="string" required>
  Exoplanet name (max 100 characters)
</ParamField>

<ParamField body="tipo" type="string" required>
  Classification type (max 50 characters)
  
  Examples: "Súper Tierra", "Gigante Gaseoso", "Neptuniano"
</ParamField>

<ParamField body="distanciaAnosLuz" type="double" required>
  Distance from Earth in light-years (must be >= 0)
</ParamField>

<ParamField body="masaRelativaJupiter" type="double" required>
  Mass relative to Jupiter (must be >= 0)
</ParamField>

<ParamField body="descubiertoAnio" type="integer" required>
  Year of discovery (1990-2100)
</ParamField>
